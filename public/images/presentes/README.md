# Imagens dos presentes

Coloque aqui as fotos de cada presente. Os nomes dos arquivos precisam ser
exatamente estes — eles são gravados na coluna `image_url` da tabela `gifts`.
Formato sugerido: JPG, proporção ~4:3, largura de 800px ou mais.

Enquanto o arquivo não existir, o card mostra apenas o título do presente.
